 # 「算力叠加增值」开源项目方案

## 一、项目定位

本项目是一个面向技术验证的开源软件原型，核心目标是探索“个人/商家闲置算力与厂商基础算力的叠加调度机制”。通过分布式调度框架，将碎片化的算力资源整合为统一的计算池，为AI训练、数据处理等场景提供补充算力。

项目本身不设商业化目标，不追求盈利，不进行规模化运营，仅作为技术探索与开源协作的载体。


## 二、项目属性

### 2.1 开源承诺
- 核心代码全量开源，采用 **MIT 许可证**
- 任何人可自由使用、修改、分发，无需授权或付费

- 社区驱动开发，欢迎贡献者参与

### 2.2 非商业化原则

## 四、技术架构

```
[用户] → [网页界面] → [调度中心] → [节点集群] → [任务结果]
```

### 4.1 组件说明
- **网页界面**：任务提交入口，状态查看面板
- **调度中心**：负责任务队列、节点匹配、结果路由
- **节点客户端**：部署于贡献者设备，执行实际计算任务

### 4.2 核心机制
- 闲置检测：仅当设备空闲时参与任务调度
- 文件隔离：用户数据与任务数据物理分离
- 自动清理：任务完成后彻底删除临时文件
